# Django Course: My Web Project

A simple desktop-oriented Django web application created for learning core Django concepts: models, views, templates, static and media files, forms, and basic e-commerce flows (products, cart, checkout, orders, contact, tracking).

This repository is a learning project built while following a Django course. It's intentionally small and opinionated for clarity rather than production readiness.

## Features

- Basic product listing and detail pages
- Search and product filtering
- Shopping cart and checkout flow (simplified)
- Order tracking and contact form
- Admin site for managing products and orders
- Static and media handling for local development

## Tech stack

- Python 3.8+
- Django 3.2 / 4.x (adjust to your local Django version)
- SQLite (default, file: db.sqlite3)
